# flask-mongodb-crud
Deploy simple CRUD website using MongoDB and Python-Flask on SAP Cloud Platform

Youtube video:
https://youtu.be/nW-agREHs7g

Command cloud foundry:

cf login

cf marketplace

cf create-service mongodb v4.0-dev mongodb-crud

cf services
